# Bug: 预置公告 source_url 全部无效

## Bug 报告

- **现象**：预置数据中 24 条公告的 source_url 全部无效，指向政府首页/栏目列表页而非具体公告
- **复现步骤**：选择人才引进 → 岗位匹配 → 点击任一公告 → "查看公告原文"
- **复现率**：必现（24/24 = 100%）
- **影响范围**：所有使用人才引进预置数据的用户
- **影响平台**：双平台
- **根因**：预置数据为 AI 生成的虚构数据，URL 仅取了各地人社局的通用页面地址

## 修复策略

为每个城市搜索真实的人才引进/事业单位招聘公告：
1. 从各地人社局官网搜索真实公告
2. 获取具体公告页面 URL
3. 提取真实的标题、内容、岗位信息
4. 替换 `assets/data/rencaiyinjin_policies_preset.json`
5. 验证每个 URL 可访问且内容匹配
