# 成语整理 (idiom_reference)

## 核心需求
整理言语理解模块中选词填空部分的成语释义以及成语在人民日报官网的多个使用例句。

## 确认方案

### 锁定决策

- 数据来源：用户点击"一键整理"→ 扫描题库提取成语 → 爬取百度汉语释义 + 人民日报例句
- 例句范围：2020-2025 年，按年份降序，每个成语最多 5 条
- 成语识别：规则匹配选项中恰好 4 个汉字的词
- 存储：SQLite 本地缓存，已整理的不重复抓取
- 入口：独立"成语整理"页面（手风琴展开）+ QuestionCard 解析区折叠显示
- 范围：仅选词填空（content 含 ___）
- 爬取限速：≥2s 间隔

### 验收标准
- [mechanical] 新表、新文件、Provider 注册、QuestionCard 集成
- [test] flutter analyze 零错误，flutter test 全通过
- [manual] 成语整理页面和 QuestionCard 成语释义展示
